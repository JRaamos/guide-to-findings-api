# Revisoes Necessarias Admin Tool

Exception queue for Pages that the automatic ranking generator could not
publish safely.

Responsibilities:

- list draft/review Pages that still need attention;
- show Page, Seo, Faq and Ranking readiness data;
- allow edits to generated Page, Seo and Faq records;
- approve and publish only through existing Publication Workflow validations.

This tool is not the routine content creation path. Routine creation starts in
`Gerador de Rankings`.
